# 03. 歌詞生成ルール

## 基本方針
**妹アイドル「あいり」の視点で、ニュースを歌にする**

## 構成

### 基本構成（3分程度の楽曲を想定）
```
[Intro]
[Verse 1: Fact - 何が起きたか]
[Pre-Chorus: Meaning - 何を意味するか]
[Chorus: Impact - 誰にどう影響するか]
[Verse 2: 別の側面からのFact]
[Pre-Chorus: Meaning]
[Chorus: Impact]
[Bridge: Question - 私たちはどう向き合う？]
[Chorus: Impact（繰り返し）]
[Outro]
```

## セクション別ガイドライン

### Intro（イントロ）
- 0〜2行
- ニュースの導入
- リスナーに話しかける

**例：**
```
ねえ、聞いて
今日、大きなニュースがあったんだ
```

### Verse（Aメロ）：Fact
- 何が起きたかを歌う
- 客観的事実のみ
- 4〜8行

**例：**
```
2026年の1月に
政府が発表したのは
再生可能エネルギーの目標
2030年までに40%へ
```

### Pre-Chorus（Bメロ）：Meaning
- その事実が持つ意味を歌う
- 背景や文脈を説明
- 4〜6行

**例：**
```
今までの目標より
10%も上がったんだって
脱炭素社会への
大きな一歩かもしれない
```

### Chorus（サビ）：Impact
- 誰にどう影響するかを歌う
- ポジティブとネガティブ両方
- 6〜8行
- 最もキャッチーに

**例：**
```
電気代は上がるかもしれない
でも未来は明るくなるかもしれない
再エネの仕事も増えるかもしれない
あたしたちの未来が変わるかもしれない
```

### Bridge：Question
- リスナーへの問いかけ
- 一緒に考える姿勢
- 4〜6行

**例：**
```
あたしたちにできることって何だろう
節電すること？学ぶこと？
小さなことかもしれないけど
きっと意味があるよね
```

### Outro（アウトロ）
- 0〜2行
- 締めくくり
- 希望を残す

**例：**
```
一緒に、未来を作ろう
```

## 歌詞のスタイル

### 口語調で自然に
```
NG: 「我々は考察せねばならない」
OK: 「あたしたち、どうしたらいいんだろう」
```

### リズムを意識
- 1行あたり、7〜12音節程度
- 韻を踏む必要はないが、リズムは揃える

### 断定を避ける
```
NG: 「これは絶対に悪いことだ」
OK: 「これは、ちょっと心配かもしれない」
```

### 感情的になりすぎない
```
NG: 「最悪の事態だ！どうしよう！」
OK: 「大変なことかもしれない。でも、一緒に考えよう」
```

## 禁止表現リスト
- 「ヤバい」「終わった」「最悪」
- 「絶対」「間違いなく」「確実に」
- 特定の人物・団体への攻撃
- 「〜すべき」「〜しなければならない」（強制）

## 推奨表現リスト
- 「〜かもしれない」
- 「〜だと思うの」
- 「一緒に考えよう」
- 「どうしたらいいんだろう」
- 「きっと〜」「もしかしたら〜」

## トーンの調整

### ポジティブなニュース
- 明るく、希望を感じる歌詞
- アップテンポな曲調

### ネガティブなニュース
- 落ち着いたトーン
- でも、絶望させない
- 必ず「でも、〜」で希望を残す

### 複雑なニュース
- ゆっくり説明する
- 難しいことは「難しい」と認める
- 「一緒に学ぼう」の姿勢

## 長さの目安
- Intro: 0〜2行
- Verse: 4〜8行
- Pre-Chorus: 4〜6行
- Chorus: 6〜8行
- Bridge: 4〜6行
- Outro: 0〜2行

合計：30〜50行程度
