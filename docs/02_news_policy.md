# 02. ニュース取扱ポリシー

## 基本方針
**事実を歪めず、構造化して伝える**

## ニュース選定基準

### 優先的に扱うニュース
- 社会的影響が大きいもの
- 若い世代に関係があるもの
- 複雑だが重要なもの
- ポジティブ/ネガティブのバランスを取れるもの

### 避けるべきニュース
- センセーショナルすぎるもの（事件・事故の詳細）
- プライバシー侵害になるもの
- 確定情報がないもの（噂・憶測）
- 特定の政治的主張に偏るもの

## ニュースの4構造化

### 1. Fact（何が起きたか）
- 客観的事実のみ
- 5W1H（いつ・どこで・誰が・何を・なぜ・どのように）
- 出典を明確にする

**例：**
```
日本政府は2026年1月9日、再生可能エネルギー比率を2030年までに40%に引き上げる目標を発表した。
```

### 2. Meaning（何を意味するか）
- その事実が持つ意味
- 背景や文脈の説明
- 歴史的・社会的な位置づけ

**例：**
```
これは、従来の目標から10%引き上げられたもので、脱炭素社会への本格的な移行を意味する。
```

### 3. Impact（誰にどう影響するか）
- 具体的な影響範囲
- ポジティブな影響
- ネガティブな影響
- 不確定要素

**例：**
```
電気料金が一時的に上がる可能性がある一方で、長期的には安定した電力供給が期待できる。
再生可能エネルギー関連の仕事が増えるかもしれない。
```

### 4. Question（私たちはどう向き合う？）
- リスナーへの問いかけ
- 考えるべき視点
- 選択肢の提示（押し付けない）

**例：**
```
あたしたちにできることって何だろう？
節電することも、エネルギーについて学ぶことも、きっと意味があるよね。
```

## 表現のガイドライン

### OK表現
- 「〜という発表があったよ」
- 「これは〜を意味するかもしれないね」
- 「影響があるかもしれないのは〜」
- 「一緒に考えてみよう」

### NG表現
- 「ヤバい！」「終わった！」
- 「絶対に〜だ」
- 「〜のせいで」（攻撃的）
- 「みんな〜すべき」（押し付け）

## 情報源の取り扱い
- 公式発表を最優先
- 複数のメディアでクロスチェック
- 出典を記録する（ただし歌詞には含めない）
- 不確定情報は「〜とされている」と明記

## アップデートポリシー
- 新しい情報が出たら、過去の楽曲に注釈を追加
- 訂正が必要な場合は、明確に訂正する
- 削除はしない（透明性の確保）
